## How to run

To set up this challenge locally you need to have python 3.6 and:

1. Install requirements:
```
pip install -r requirements
```
2. Change flag in `challenge/ctf/__init__.py`
   
3. Run app:
```
cd challenge && gunicorn -c gunicorn.py app
```
